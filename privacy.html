Política de Privacidade

Esta aplicação foi criada para uso pessoal e privado com a finalidade de automatizar o envio de vídeos para a conta do proprietário na plataforma TikTok.

1. Coleta de Dados

Coletamos apenas os dados necessários para autenticação com a API do TikTok, como tokens de acesso. Nenhum dado pessoal adicional é armazenado ou processado.

2. Uso dos Dados

Os dados obtidos são utilizados exclusivamente para realizar publicações na conta autenticada. Nenhum dado é compartilhado com terceiros.

3. Armazenamento

Os tokens de acesso são armazenados temporariamente e utilizados apenas durante a sessão ativa. Não há retenção prolongada nem banco de dados público.

4. Direitos do Usuário

Como o único usuário é o próprio desenvolvedor, todas as permissões e revogações podem ser feitas diretamente pela conta TikTok.

5. Segurança

Medidas básicas de segurança são aplicadas para proteger o uso dos tokens. Recomendamos que tokens sejam renovados periodicamente.

Ao utilizar este sistema, você concorda com esta Política de Privacidade.
